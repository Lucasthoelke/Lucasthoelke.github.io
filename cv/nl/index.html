AVG voldoenende data opslag

Vo De Vallei

Nadat de nieuwe AVG wetgeving tot kracht kwam moest er een oplossing worden verzonnen om veilig en goedkoop leerling data op te slaan. Ik heb hiervoor een server gebouwd die op locatie staat. Deze server draait onder Promox VE en heeft een VM met Nextcloud erop. Deze VM heeft toegang tot 2 WD Reds in RAID en wordt dagelijks off-site geëncrypt en gebackupped. Het interface van Nextcloud draait achter een reverse proxy (NGINX) met SSL.

Canyon Framework

Privé source

Toen ik bezig was met prototypes voor de uren Registratie (Vo De Vallei) heb ik mijn eigen framework gebouwd in PHP. Features waren o.a. Modules, Geavanceerde Exception Management, Logger, Geautomatiseerde API, Permission systeem, Session systeem met permissions, User systeem en nog een hoop kleine dingen. Ik heb uiteindelijk afgezien dit framework daadwerkelijk te gebruiken bij de uren registratie omdat een eigen framework onderhouden teveel tijd kostten en er een paar ontwerp fouten in zaten. Ik heb het wel gebruikt voor wat kleine test projecten.
